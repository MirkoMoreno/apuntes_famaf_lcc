# Introducción (CA)

En este apartado se cubren los contenidos de Cálculo Algebraico, donde se incluyen los distintos conjuntos numéricos, las operaciones básicas, las propiedades de las operaciones, los exponentes y radicales, las expresiones algebraicas, ecuaciones, entre otros. 

El objetivo de este apartado es desarrollar el pensamiento algebraico, que es la capacidad de representar, analizar y resolver situaciones que involucran relaciones cuantitativas, mediante el uso de símbolos y expresiones algebraicas.
